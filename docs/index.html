FamiStep アカウント作成・削除手順

アカウントの作成方法

  1. 下タブから「マイページ」をタップ
  2. 右上の歯車マークをタップ
  3. 「アカウント」をタップ
  4. 「アカウントの登録」をタップ
  5. メールアドレスとパスワードを入力して、「登録」ボタンを押下
  6. メールアドレス確認メールの中のリンクをタップ
  7. アカウント作成完了

アカウントの削除方法

  1. 下タブから「マイページ」をタップ
  2. 右上の歯車マークをタップ
  3. 「アカウント」をタップ
  4. 「退会」をタップ
  5. アカウント削除完了

データの取り扱いについて

アカウント削除後、以下のデータが完全に削除されます：

  • ユーザープロフィール情報
  • アカウントに関連するすべてのデータ

※データは削除後30日間保持され、その後完全に削除されます。
